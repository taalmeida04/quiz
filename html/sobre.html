Sobre o Quiz

O Jurassic Park Quiz foi criado como parte de um projeto escolar da disciplina de Interfaces Web II, com o objetivo de desenvolver um sistema de quiz utilizando tecnologias web e aplicar, na prática, os conhecimentos adquiridos em sala de aula. O projeto foi desenvolvido por alunos do 2º ano do Ensino Médio do curso Técnico em Informática para Internet, unindo criatividade, programação e design para criar uma experiência interativa inspirada no universo de Jurassic Park. Além de avaliar conhecimentos sobre a franquia, o quiz demonstra a aplicação de conceitos de desenvolvimento web, interface do usuário e experiência do usuário.
